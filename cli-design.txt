1) Implemented
vlt cfg purge: remove vault
vlt cfg reset-pass: reset master password

vlt pass get -S aws -I dvdagreat: get credential secret string where service is aws and identifier is dvdagreat
vlt pass get: get credential secret but select service & identifier via interactive terminal prompts
vlt pass set -S aws -I dvdagreat: set credential secret string where service is aws and identifier is dvdagreat. Uses a hidden text input
vlt pass set: set credential secret but select service & identifier via interactive terminal prompts. Uses a hidden text input
vlt pass rm -S aws -I dvdagreat: removes credential secret where service is aws & identifier is dvdagreat.
vlt pass rm: removes credential secret but select service & identifier via interactive terminal prompts.


2) Commands not yet implemented
vlt ident list: print a list of identifiers with their associated service
vlt ident list -S aws: list all identifiers in service aws
vlt ident list -S aws -S facebook: list all identifiers in services aws & facebook
vlt ident find -S aws -S facebook -N dvdagreat: check if provided identifier exists in service aws & facebook
vlt ident set -S aws -N dvdagreat: set identifier name where service is aws and identifier is dvdagreat (essentially rename identifier)
vlt ident rm -S aws -N dvdagreat: same as removing password

vlt srv list: lists all the services
vlt srv find -R *aws* : fuzzy search services
vlt srv rm -N aws: removes service which matches the name aws including all the indentifiers/credentials
vlt srv rm: remove service but select service via interactive terminal prompts
